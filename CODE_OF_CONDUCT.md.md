# Contributor Code of Conduct

As contributors and maintainers of OSO Tools, we pledge to make participation in our project a harassment-free experience for everyone.

We value:
- Respectful and constructive communication  

- Collaboration and learning  

- Welcoming new contributors of all skill levels  

  

Unacceptable behavior includes:
- Harassment, discrimination, or hate speech  

- Personal attacks or trolling  

- Sharing others’ private information without consent  

  

If you experience or witness any unacceptable behavior, please open a confidential issue or contact a maintainer.

Let’s build great tools and a great community together.
