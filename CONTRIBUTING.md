# Contributing to OSO Tools

Thanks for your interest in contributing!
We’re building a collection of open-source tools and scripts that make sysadmin life easier, and your ideas, tools, and improvements are welcome.



## How to Contribute

1. **[Fork](https://github.com/michaelkedey/oso-tools/fork)** this repository  
2. **Create a new branch** for your feature or fix  
   ```bash
   git checkout -b my-new-tool

3. **Write your tool**

4. **Test it** and make sure it works cleanly on Linux

5. **Commit and push your changes**

   ```bash
   git commit -m "Add: new cleanup script"
   git push origin my-new-tool
   ```

6. **Open a Pull Request (PR)**,  describe what your tool does and how to use it

7. We’ll review your PR and merge it if everything looks good!
