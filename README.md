### What is OSO-Tools?

OSO Tools **(Open Source Tools)** is a growing, community-driven toolbox of utilities built by sysadmins, for sysadmins.

These tools aim to simplify:
- Server management and maintenance  

- Automation of daily tasks  

- Monitoring, cleanup, and troubleshooting   among other things

  

### Example Tools

| Tool              | Description                       | Language |
| ----------------- | --------------------------------- | -------- |
| `cleanup.sh`      | Cleans temp and log files safely  | Bash     |
| `uptime_check.py` | Checks uptime and sends alerts    | Python   |
| `backup_sync.sh`  | Automates backups between servers | Bash     |
